This file is for creating cities only (no districts on this upload).

Required columns:
  city_name   City display name (e.g. الرياض).
  city_code   Short unique code (e.g. RYD-EX); used to match or create the city.

Optional column (omit for cities-only; or include with values to create districts in the same upload):
  district_name   District / neighborhood (الحي) under that city.

You can also use a three-column CSV (city_name, city_code, district_name) with district_name left blank on each row — same effect as two columns.

Import districts in a second step with POST /api/admin/districts/import_csv and districts_import.csv (city_id after cities exist).
